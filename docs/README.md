ビーウィズ株式会社で約 4 年間、コールセンター向け Web アプリケーションの開発を担当しました。React によるフロントエンド開発を中心に、画面の設計から実装まで携わりました。バックエンド（Laravel）やデータベースの設計も担当していました。
